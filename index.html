Calculadora

Um app de calculadora.

Um app simples de calculadora feito utilizando React Native onde se é possível efetuar as operações básicas de uma calculadora!

  • Tecnologia(s): React Native e JavaScript.

Market Place

Um sistema de compras online com integração de uma API de pagamento.

O sistema consiste em uma loja online onde se é possível gerenciar sua loja, adicionar itens no carrinho e efetuar o pagamento dentro do site através do uso da API de pagamento do PagSeguro. O sistema foi todo desenvolvido utilizando o Framework Laravel do PHP, juntamente com todas as ferramentas que o Framework proporciona

  • Tecnologia(s): Laravel, Bootstrap, JavaScript, MySQL.

Simple CRUD

Biblioteca desenvolvida em PHP para manipulação de CRUD.

O Projeto consiste em ser um simplificador para as operações de um CRUD, Create, Read, Update e Delete, onde precisa-rá apenas utilizar os métodos disponíveis pela lib para poder efetuar todas essa operações sem ficar precisando se matar para escrever query no meio do PHP!

  • Tecnologia(s): PHP e Composer.

DS Deliver

API Rest Java com integração front web e mobile.

O projeto foi desenvolvido durante o evento online e gratuito disponibilizado pela DevSuperior chamado "Semana Spring React". O objeto do projeto era permitir que pessoas pudessem solicitar a entrega de um alimento através do app. Foi desenvolvido uma API Rest usando Spring Boot, framework Java para desenvolvimento web, juntamente com o desenvolvimento das interfaces mobile e web da aplicação, cujo as ferramentas utilizadas foram React e React Native. O projeto também consta com o consumo de uma API de geolocalização !

  • Tecnologia(s): Java, Spring Boot, React, React Native, MySQL

GEM

Sistema desenvolvido para ONG Estrela da Mama.

Eu participei do desenvolvimento do sistema GEM para uma ONG de Praia Grande como trabalho voluntário, o sistema foi desenvolvido para atender a necessidade da presidente da ONG que era a gestão de todos os afiliados. Eu juntamente com um grupo de amigos, levantamos os requisitos para o desenvolvimento da aplicação e ao final entregamos o sistema a presidente da forma que ela necessitava!

  • Tecnologia(s): PHP, Composer, HTML, CSS, JavaScript e MySQL

Heat Tags

Evento NLW especial da Rocketseat para desenvolver o app de nuvem de palavras usados no Do While(outro evento da Rocketseat).

Esse projeto foi desenvolvido no evento NLW da Rocketseat, mas diferentes dos eventos anteriores esse foi um evento especial onde foi desenvolvido uma unica aplicação utilizando uma grande variedades de tecnologias. A primeira delas foi o NodeJs usado para desenvolver a API da aplicação com o direito ao consumo da API de autenticação do GitHub. Após isso foi desenvolvido as interfaces utilizando React e React Native respectivamente. E por ultimo mas não menos importante o desenvolvimento de um micro serviço utilizando Elixir e que é uma ferramente que venho estudando e tenho gostado muito!

  • Tecnologia(s): Node, React, React Native, Elixir, Prisma e Postgres
